# STYLED_TEXT_GENERATOR

A simple web tool to convert plain text into stylish Unicode formats. Supports multiple input boxes, a variety of font styles and easy copy-to-clipboard functionality. Ideal for creating visually appealing text for social media posts, messages, or content creation.

## Features

* Convert text into multiple Unicode styles:

  * Bold (Serif)
  * Bold Sans-Serif
  * Italic
  * Bold Italic
  * Monospace
  * Script
  * Fraktur
  * Double Struck
* Multiple input boxes for simultaneous conversions.
* Copy styled text to clipboard with a single click.
* Responsive and modern UI with gradient background and clean design.
* Lightweight, no backend required (fully client-side).
